\chapter*{ЗАКЛЮЧЕНИЕ}
\addcontentsline{toc}{chapter}{ЗАКЛЮЧЕНИЕ}
В работе получены следующие основные результаты:
\begin{enumerate}
	\item Исследована математическая модель вкраплений в цепь Маркова 1-го порядка.
	\item Получены точные значения вероятностей для всевозможных шаблонов \textit{l}-граммы при \textit{l=2,3,4}.
	\item Получены асимптотические оценки первого порядка для энтропии \textit{l}-граммы при \textit{l=2,3,4}.
	\item Проведён линейный дискриминантный анализ на основании асимптотических оценок для энтропии 3-граммы и 4-граммы.
	\item Проведён ряд компьютерных экспериментов.
	
	
  
\end{enumerate}
