\begin{frame}
\frametitle{Введение}
\begin{footnotesize}
Генераторы случайных чисел:
\begin{itemize}
  \item ГПСЧ
  \item ГПСЧ + источник энтропии (случайности) = ГСЧ (физический, аппаратный)
\end{itemize}
В работе описаны свойства, которыми источник энтропии должен обладать, чтобы быть пригодным для использования в генераторах случайных битов, проведен обзор существующих физических источников случайности. Описаны статистические тесты, используемые для определения качества источника энтропии по данным, полученным от него. Разработано универсальное программное обеспечение для тестирования источников случайности.
\end{footnotesize}
\end{frame}

\begin{frame}
\frametitle{Энтропия}
\begin{footnotesize}
Двоичная энтропия выходной последовательности источника случайности:
\begin{equation}
    H=-\sum_{i=1}^{n} p_i \log{p_i},
\end{equation}
где $p_i$ -- это вероятность состояния $i$ из $n$ возможных состояний. В случае, когда генератор случайных чисел обеспечивает $k$-битный результат, $p_i$ -- вероятность того, что выходное значение будет равняться $i$, где $0 \leq i <  2^k$. Таким образом, для идеального генератора случайных чисел $p_i=2^{-k}$ и энтропия выхода равна $k$ битам. Это означает, что все возможные исходы равновероятны, а информация, представленная в выходе, в среднем не может быть представлена последовательностью длиной короче, чем $k$ бит.
\end{footnotesize}
\end{frame}


\begin{frame}
\frametitle{Минимальная энтропия}
\begin{footnotesize}
Минимальная энтропия связана с наблюдениями $X$ в наихудшем случае: если $X$ имеет минимальную энтропию $m$, то вероятность наблюдения какого-либо конкретного значения не больше, чем $2^{-m}$. Пусть $x_1,\ x_2,\ \ldots,\ x_M$ -- выходы источника случайности, и пусть $p(x_i)$ есть вероятность того, что $x_i$ получен в любой момент получения образцов. Тогда минимальная энтропия выходов определяется формулой \cite{draft}:
\begin{equation}
    -\log_2 (\max\ p(x_i)).
\end{equation}
Это соответствует работе злоумышленника, пытающегося угадать выход источника шума, в наилучшем для него случае. Максимальным возможным значением минимальной энтропии случайной величины с $k$ различными значениями является $\log_2 k$, которое достигается когда случайная величина имеет равномерное распределение вероятностей, т.е., $p_1 = p_2 \ldots = p_k =1/k$.
\end{footnotesize}
\end{frame}

\begin{frame}
\frametitle{Оценка энтропии}
\begin{footnotesize}
Оценка энтропии выходных последовательностей состоит в применении специальных тестов для оценки минимальной энтропии.

Оценка энтропии осуществляется по-разному для выходных последовательностей, являющихся последовательностью н.о.р. случайных величин и не являющихся таковыми.

Проверка выходных последовательностей на н.о.р. состоит в применении специальных статистических тестов.
\end{footnotesize}
\end{frame}

\begin{frame}
\frametitle{Тесты н.о.р. Тесты перестановок}
\begin{footnotesize}

\begin{itemize}

\end{itemize}
\end{footnotesize}

\begin{frame}
\frametitle{Тесты н.о.р. Хи-квадрат тесты}
\begin{footnotesize}
Используется два различных типа хи-квадрат критериев: критерий независимости и критерий согласия.
\begin{itemize}

\item Хи-квадрат критерий независимости предназначен для обнаружения зависимости в вероятностях между соседними наблюдениями. Существует в варианте для небинарных данных и для бинарных данных.\\
\item Критерий согласия хи-квадрат предназначен для проверки согласия распределения подвыборок, сформированных из исходной выборки. Существует в варианте для небинарных данных и для бинарных данных.
\end{itemize}
\end{footnotesize}
\end{frame}


\begin{frame}
\frametitle{Оценка энтропии}
\begin{footnotesize}
\begin{itemize}
  \item Тест для оценки энтропии н.о.р. данных\\
\end{itemize}

Для оценки энтропии последовательности случайных величин, не являющихся н.о.р, используется пять статистических тестов:
\begin{itemize}
\item Тест коллизий
\item Тест частичных коллекций
\item Тест марковской зависимости
\item Тест сжатия
\item Частотный тест
\end{itemize}
В качестве оценки минимальной энтропии используется минимальная из всех оценок, полученных каждым тестом.

\end{footnotesize}
\end{frame}

\begin{frame}
\frametitle{Разработанное ПО тестирования}
\begin{footnotesize}
Было разработано программное обеспечение тестирования выходных последовательностей источников случайностей. В нем реализованы тесты, описанные в диссертации. Пакет тестов реализован в виде консольного приложения. Формат вызова: \verb">EntropySourceTesting.exe FileName SampleSize", где \verb"FileName" -- имя файла содержащего выходную последовательность в двоичном виде, \verb"SampleSize" -- количество битов представляющих одно наблюдение. Размер наблюдения технически ограничен 32 битами.\\
Пример:\\
Вызов \verb">EntropySourceTesting.exe data.bin 8" означает, что последовательность будет считываться из файла \verb"data.bin", а размер одного наблюдения -- 8 бит, т.е. наблюдения лежат в интервале $[0, 2^8 - 1]$.
\end{footnotesize}
\end{frame}


\begin{frame}
\frametitle{Результаты работы ПО}
\begin{footnotesize}
При запуске программы на последовательности осциллятора с заданными размерами наблюдения 1, 8, 16 и 32 бита получим следующие результаты:
\begin{enumerate}
  \item H = 1 при размере наблюдения 1 бит.
  \item H = 8 при размере наблюдения 8 битов.
  \item H = 15,6022308384072 при размере наблюдения 16 битов.
  \item H = 16,2983682583541 при размере наблюдения 32 битов.
\end{enumerate}
\end{footnotesize}
\end{frame}
